Lookup API
✨ Fast • Secure • Serverless

Simple Phone Lookup API

Minimal proxy endpoint for phone lookup requests. Built for speed with Vercel Serverless Functions. API key stays protected on the server.

Try Example 
// Example Request

GET /api/lookup?number=7571902689

Response

{
  "success": true,
  "result": {
      "fname":"Godrej Agrovet Limited",
      "circle":"AIRTEL UP EAST",
      "address":"Unnao, Uttar Pradesh"
  }
}

⚡ Fast

Runs on Vercel Edge Functions with very low latency.

🔒 Secure

API key remains hidden using server-side environment variables.

📦 JSON

Returns clean JSON responses suitable for apps and bots.

💻 Simple

Single endpoint with just one required query parameter.
